ğŸ§  Step 4 â€” Writing Our First AI Program

Create a file called main.py 
import os
from dotenv import load_dotenv
from openai import OpenAI

load_dotenv(override=True)
api_key = os.getenv('OPENAI_API_KEY')

# Validation checks
if not api_key:
    print("No API key found!")
elif not api_key.startswith("sk-"):
    print("API key format looks wrong.")
else:
    print("API key found!")

client = OpenAI()

# Simple request
message = "Hello, GPT! This is my first ever message to you!"
messages = [{"role": "user", "content": message}]

response = client.chat.completions.create(
    model="gpt-4o-mini", 
    messages=messages
)

print(f"Response: {response.choices[0].message.content}")

ğŸ¤– Step 5 â€” System and User Prompts

AI behavior is programmable using prompts.

System Prompt

Defines AI personality and rules. Examples:
  • You are a Python tutor
  • You are a sarcastic robot
  • You are an HR expert

User Prompt

The actual question or task. Examples:
  • Explain decorators
  • Review my resume
  • What is 2 + 2?
